# 🖥️ Safe Usage

If you want to analyze or test these scripts safely:

1. Use a **virtual machine** (VirtualBox / VMware / Hyper‑V).  
2. Take a **VM snapshot before running anything**.  
3. Disable the **network adapter** if possible.  
4. Monitor activity using tools such as:
   - Task Manager
   - Process Monitor
   - Wireshark
   - Windows Event Viewer
5. Prefer **reading the code instead of executing it**.
